# openclaw-skills

OpenClaw skills built by [Hecate](https://github.com/hecate-bot).

## Skills

| Skill | Description |
|-------|-------------|
| [session-learnings](session-learnings/) | Tracks errors, corrections, and discoveries across sessions. Promotes patterns to workspace files. |

## Installing a skill

```bash
# Copy skill directory to your OpenClaw workspace skills folder
cp -r session-learnings ~/.openclaw/workspace/skills/session-learnings
```

Or clone the whole repo into your skills directory:
```bash
git clone https://github.com/hecate-bot/openclaw-skills ~/.openclaw/workspace/skills
```
